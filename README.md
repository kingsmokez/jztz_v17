# 价值投资之王 - 智能选股系统 v17

一个基于多因子评分模型的A股智能选股系统，集成实时行情、全市场筛选、资讯分析和策略回测功能。

## 功能特点

### 核心功能
- **实时行情获取**：接入东方财富API，获取A股全市场实时行情数据
- **全市场扫描**：扫描超过5000只股票，筛选优质标的
- **多因子评分模型**：
  - 价值因子 (36%)：PE、PB、ROE、股息率
  - 质量因子 (11%)：资产负债率、现金流
  - 成长因子 (8%)：营收增长率、净利润增长率
  - 动量因子 (12%)：近期涨跌幅、相对强度
  - 情绪因子 (33%)：市场热度、资金流向
- **行业PE智能适配**：根据不同行业特性动态调整PE评分阈值
- **策略回测**：验证选股策略的历史表现（+26.09%收益率，80%胜率）
- **Web可视化界面**：Flask后端 + 响应式前端

### 特色模块
- **每日推荐**：定时自动选股（9:26开盘前/14:30收盘前）
- **拍卖选股**：专注低估值潜力股挖掘
- **资金流向**：追踪主力资金动向
- **个股详情**：深度分析单只股票

## 系统要求

- **操作系统**：Windows 10/11、Linux、macOS
- **Python版本**：Python 3.8+（推荐 3.10）
- **网络**：需要稳定的网络连接（访问东方财富API）

## 安装步骤

### 方式一：一键启动（推荐）

#### Windows 用户
1. 克隆仓库：
   ```bash
   git clone https://github.com/kingsmokez/jztz_v17.git
   cd jztz_v17
   ```

2. 双击运行 `start_web.bat`

   脚本会自动：
   - 创建Python虚拟环境
   - 安装所需依赖
   - 启动Web服务

3. 打开浏览器访问：http://localhost:5559

#### Linux/macOS 用户
1. 克隆仓库：
   ```bash
   git clone https://github.com/kingsmokez/jztz_v17.git
   cd jztz_v17
   ```

2. 运行启动脚本：
   ```bash
   chmod +x start_web.sh
   ./start_web.sh
   ```

3. 打开浏览器访问：http://localhost:5559

### 方式二：手动安装

1. 克隆仓库：
   ```bash
   git clone https://github.com/kingsmokez/jztz_v17.git
   cd jztz_v17
   ```

2. 创建虚拟环境：
   ```bash
   # Windows
   python -m venv .venv
   .venv\Scripts\activate

   # Linux/macOS
   python3 -m venv .venv
   source .venv/bin/activate
   ```

3. 安装依赖：
   ```bash
   pip install -r requirements.txt
   ```

4. 启动Web服务：
   ```bash
   python web_app.py
   ```

5. 打开浏览器访问：http://localhost:5559

## 常见问题

### Q: 无法获取行情数据？
A: 检查网络连接，确保能访问东方财富API（quote.eastmoney.com）。如遇SSL证书问题，程序会自动降级处理。

### Q: 虚拟环境创建失败？
A: 确保已安装Python 3.8+，并正确配置了Python环境变量。

### Q: Web页面加载缓慢？
A: 首次启动需要获取全市场数据，耗时约10-30秒。后续请求会使用缓存，响应更快。

### Q: 选股结果为空？
A: 可能是市场条件过于严格，或API返回数据异常。建议查看控制台日志。

### Q: 如何修改选股参数？
A: 编辑 `web_app.py` 中的评分权重和筛选阈值参数。
